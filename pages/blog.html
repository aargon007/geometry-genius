DOM is a Document Object Model & Programming Interface for HTML that defines HTML objects, properties, methods & events to get, change, add or delete HTML elements.The purpose of DOM is to maintain a standard programming interface for HTML and XML documents, which is language-independent.By using DOM we can locating elements in the document by their ID, class, tag name and other characteristics.We can also add, remove or modify HTML element and attributes.

There are various DOM methods that can be use for finding HTML elements.We can select HTML element by using getElementById() method and getElementsByClassName(), getElementsByTagName(), querySelectorAll() methods return a set of array like object. We can implement those by for loop.

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element(child), and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object.

In JavaScript, a callback function is a function that is passed as an argument to another function and is executed after some operation has been completed. Callbacks allow us to write asynchronous code, where we can continue to execute other code while waiting for a particular event to occur.
